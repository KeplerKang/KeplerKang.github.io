# 设计师作品集网站 - 测试说明

## 测试目标

验证网站从单页面结构成功转换为多页面结构，确保所有导航链接正常工作，各页面内容正确显示。

## 测试环境

- 浏览器：Chrome、Firefox、Safari、Edge
- 设备：桌面电脑、平板、手机

## 测试步骤

### 1. 首页测试 (index.html / home.html)

1. 打开 `index.html` 或 `home.html`
2. 验证页面显示正确的内容：
   - 导航栏显示且包含所有菜单项
   - Hero区域显示个人简介和图片
   - 最新作品预览区域显示3个作品卡片
   - 页脚显示正确的版权信息和社交媒体链接
3. 点击导航栏中的各个链接，验证是否跳转到正确的页面：
   - Home → index.html 或 home.html
   - Works → works.html
   - About → about.html
   - Contact → contact.html
4. 点击"View My Work"按钮，验证是否跳转到作品列表页
5. 点击"About Me"按钮，验证是否跳转到关于我页面
6. 点击作品卡片上的"View Project"链接，验证是否跳转到对应的作品详情页
7. 点击"View All Works"按钮，验证是否跳转到作品列表页

### 2. 作品列表页测试 (works.html)

1. 打开 `works.html`
2. 验证页面显示正确的内容：
   - 导航栏显示且包含所有菜单项
   - 页面标题" My Work"显示正确
   - 分类筛选按钮显示正确
   - 作品网格显示所有6个作品卡片
3. 测试分类筛选功能：
   - 点击"All"按钮，验证显示所有作品
   - 点击"UI/UX"按钮，验证只显示UI/UX类作品
   - 点击"Branding"按钮，验证只显示Branding类作品
   - 点击"Web"按钮，验证只显示Web类作品
   - 点击"Print"按钮，验证只显示Print类作品
4. 点击任意作品卡片上的"View Project"链接，验证是否跳转到对应的作品详情页

### 3. 作品详情页测试 (works/work1.html 到 works/work6.html)

1. 打开任意一个作品详情页，例如 `works/work1.html`
2. 验证页面显示正确的内容：
   - 导航栏显示且包含所有菜单项
   - 作品标题、分类标签和简介显示正确
   - 主要图片显示正确
   - 项目概述和设计过程等详细信息显示正确
   - 项目成果部分显示正确
   - 相关项目推荐显示正确
3. 测试图片查看器功能：
   - 点击主要图片，验证图片查看器是否弹出
   - 点击图片查看器中的关闭按钮，验证是否关闭
   - 点击图片查看器背景，验证是否关闭
   - 按ESC键，验证是否关闭图片查看器
4. 点击"Back to All Works"按钮，验证是否返回作品列表页
5. 点击相关项目中的"View Project"链接，验证是否跳转到对应的作品详情页

### 4. 关于我页面测试 (about.html)

1. 打开 `about.html`
2. 验证页面显示正确的内容：
   - 导航栏显示且包含所有菜单项
   - 页面标题"About Me"显示正确
   - 个人照片显示正确
   - 个人简介显示正确
   - 技能标签显示正确
   - 经验和教育信息显示正确

### 5. 联系方式页面测试 (contact.html)

1. 打开 `contact.html`
2. 验证页面显示正确的内容：
   - 导航栏显示且包含所有菜单项
   - 页面标题"Get In Touch"显示正确
   - 联系表单显示正确
3. 测试联系表单功能：
   - 填写表单并点击"Send Message"按钮
   - 验证是否显示提交成功消息
   - 验证表单是否重置

### 6. 响应式设计测试

1. 在不同设备上测试所有页面，包括：
   - 桌面电脑（大屏幕）
   - 平板（中等屏幕）
   - 手机（小屏幕）
2. 验证在各种屏幕尺寸下：
   - 导航栏正确显示（桌面端显示完整导航，移动端显示汉堡菜单）
   - 点击移动端汉堡菜单按钮，验证菜单是否展开和收起
   - 页面内容布局合理，没有重叠或显示不全的问题
   - 图片和文字大小适应不同屏幕尺寸

### 7. 动画效果测试

1. 测试所有页面的动画效果：
   - 页面加载时的元素淡入效果
   - 滚动时的元素淡入效果
   - 作品卡片的悬停效果
   - 导航栏的滚动效果
   - 回到顶部按钮的显示和隐藏效果

## 预期结果

所有测试步骤都应顺利完成，没有错误或异常。导航链接应正确指向对应的HTML文件，页面内容应正确显示，功能应正常工作。
