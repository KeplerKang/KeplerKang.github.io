# 设计师作品集网站

这是一个现代化的设计师作品集网站，采用多页面结构设计，展示设计师的个人信息、作品和联系方式。

## 项目结构

```
designer-portfolio/
├── index.html          # 首页（与home.html内容相同）
├── home.html           # 首页
├── works.html          # 作品列表页
├── about.html          # 关于我页面
├── contact.html        # 联系方式页面
├── works/              # 作品详情页目录
│   ├── work1.html
│   ├── work2.html
│   ├── work3.html
│   ├── work4.html
│   ├── work5.html
│   └── work6.html
├── README.md           # 项目说明文档
└── TEST_INSTRUCTIONS.md # 测试说明文档
```

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- GSAP 动画库
- Font Awesome 图标库

## 功能特点

1. **响应式设计**：适配桌面端、平板和移动端
2. **多页面结构**：各功能模块分离到独立的HTML文件
3. **作品展示**：包含作品列表页和详情页
4. **动画效果**：使用GSAP实现平滑过渡和交互动画
5. **联系表单**：用户可以发送消息

## 如何使用

1. 克隆或下载项目到本地
2. 直接打开 `index.html` 或 `home.html` 即可访问网站
3. 通过导航栏在不同页面之间切换

## 页面说明

- **首页**：展示个人简介和最新作品预览
- **作品列表页**：展示所有作品，支持分类筛选
- **作品详情页**：展示单个作品的详细信息，包括项目概述、设计过程、成果等
- **关于我**：展示个人背景、技能和经验
- **联系方式**：提供联系表单和社交媒体链接

## 注意事项

- 确保所有页面的导航链接正确指向对应的HTML文件
- 图片资源使用了外部链接，确保网络连接正常
- 联系表单目前只是前端展示，实际提交功能需要后端支持
